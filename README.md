### RPA 개요
- Digital Workforce 는 사람이 컴퓨터에서 작업하는 업무를 그대로 모방하여 수행하는 Software 로봇
- 자동화 업무 대상 : 정형화된 규칙, 반복적인 업무, 다수의 인원이 필요한 업무, 노동 절약
- 작동원리 : 개발 봇으로 정형화되고 반복적인 업무를 스크립트로 작성, 작성된 스크립트를 사용자 PC에서 실행

### RPA 장점
 1. 직원 업무시간 절감
 2. 업무 생산성 향상
 3. 업무 정확도 향상
 4. 가치 있는 업무에 집중
